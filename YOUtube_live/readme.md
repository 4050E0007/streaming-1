
# Youtube直播

## 步驟一:取得Youtube金鑰
```
1.使用自己帳號登入Youtube
2.進入創作者工作室
![Alt text][youtube_0.jpg]
3.點選即時串流
4.複製串流金鑰
```

## 步驟二:設定OBS
```
5.先在OBS來源中，點選+增加需要的畫面

==>我們這次選擇
顯示器擷取(當前電腦使用畫面)
視訊擷取裝置(camera)

6.點選設定 -> 串流==>選擇Youtube live==>然後把複製的金鑰貼上
```

## 步驟三:開始直播
```
7.在OBS中按下開始串流，剛剛打開的Youtube頁面就會出現了

8.按開始直播就好了
```
